# DiarioGarage

Web app per tenere sotto controllo scadenze e costi del proprio veicolo.

## Stack
- Next.js (App Router) + TypeScript
- Tailwind + Shadcn UI
- Prisma + Postgres

## Avvio locale
```bash
npm run dev
```

## Migrazioni Prisma
```bash
npx prisma migrate dev --name <nome_migrazione>
npx prisma generate
```

## Note
- Tema system light/dark, accent viola.
